About the Project
Study Tracker Candles is a motivation-focused timer application designed to help users set and visually track their weekly study goals. Each concentrated focus session is represented by a digital candle that realistically melts as time progresses. This approach leverages visual progress feedback to help users manage their time effectively and boost motivation.
